# KLS-Furniture

A database-driven application for a **furniture rental store**, built as the Group 1 project for CS6232.

This repository contains the source code, documentation, and elements for designing and implementing a rental management system that supports customers renting furniture items, tracking inventory, handling rentals/returns, and administrative functions.

> **Note**: This is a group project for the University of West Georgia (CS6232). See the [Wiki](https://github.com/lfentress-UWG/KLS-Furniture/wiki) for detailed project information, including the full product backlog, initial UI mockups, and database schema.

## Project Purpose

The goal is to meet the requirements of the CS6232 Group Project by creating a functional, database-backed application for managing a furniture rental business. Key features will include user authentication, item browsing/rental, inventory management, rental tracking, and basic reporting.
